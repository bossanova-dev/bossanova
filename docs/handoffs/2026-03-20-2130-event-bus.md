### Implementation Notes

- `Bus` struct uses `sync.RWMutex` ‚Äî RLock for Publish (multiple concurrent publishers), full Lock for subscribe/unsubscribe/close
- Subscribers tracked via `map[*subscriber]struct{}` for O(1) add/remove
- Each subscriber gets a buffered channel of size 64 (matching poller pattern from `session/poller.go`)
- `Subscribe(ctx)` spawns a goroutine that auto-unsubscribes when the context is cancelled
- `Publish` is non-blocking: if a subscriber's buffer is full, the event is dropped and a warning is logged (no backpressure)
- `Close` is idempotent, cancels all subscriber contexts and closes all channels
- `Subscribe` after `Close` returns an immediately-closed channel (safe to range over)
- Uses proto `EventNotification` type directly ‚Äî no wrapper types needed
- Import alias: `bossanovav1 "github.com/recurser/bossalib/gen/bossanova/v1"`
