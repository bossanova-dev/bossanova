### Learnings & Notes

- Go 1.25 supports `omitzero` JSON struct tag for nested structs — `omitempty` has no effect on structs and triggers a lint warning
- AutopilotConfig uses `omitzero` (not `omitempty`) on both the Skills field and the Settings.Autopilot field so that zero-value configs are omitted from serialized JSON
- SkillName returns empty string for unknown step names (falls through to defaultSkills map lookup which returns "")
- The config package is at `lib/bossalib/config/` — go commands must run from `lib/bossalib/` directory
